# redis

Единый клиент для работы с Redis. На данном этапе пока нужна для Server Sent Events с помощью механизма pub/sub в сервисе `api`.

## Установка

Библиотеку можно установить добавлением её в зависимости вашего проекта:

```json
{
  "dependencies": {
    "@career-ai/queue": "workspace:*"
  }
}
```

Затем нужно установить эту зависимость:

```bash
bun install
```

После этого, при изменениях в библиотеке, изменения будут автоматически отражаться в использующих её сервисах.

Если сервис использует `@career-ai/redis`, то в переменные окружения `.env` нужно добавить следующие переменные:

```env
REDIS_HOST=your_redis_host
REDIS_PORT=your_redis_port
```
