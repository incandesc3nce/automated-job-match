# queue

Библиотека для управления очередями задач. Позволяет создавать и обрабатывать задачи асинхронно, обеспечивая надежное выполнение и не допуская перегрузку системы, преимущественно в сервисе `llm-worker`.

## Установка

Библиотеку можно установить добавлением её в зависимости вашего проекта:

```json
{
  "dependencies": {
    "@career-ai/queue": "workspace:*"
  }
}
```

Затем нужно установить эту зависимость:

```bash
bun install
```

После этого, при изменениях в библиотеке, изменения будут автоматически отражаться в использующих её сервисах.

Если сервис использует `@career-ai/queue`, то в переменные окружения `.env` нужно добавить следующие переменные:

```env
REDIS_HOST=your_redis_host
REDIS_PORT=your_redis_port
```

## Использование

Если необходимо создать новую очередь, можно использовать следующий код:

```typescript
type MyQueuePayload = {
  name: string;
  age: number;
};

export const myQueue = new Queue<MyQueuePayload>('my-queue', {
  connection: connectionOptions,
});
```
