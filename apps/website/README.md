# website

Это фронтенд-приложение на Next.js, которое взаимодействует с API для предоставления пользователю интерфейса для создания резюме и получения рекомендаций по вакансиям.

## Технологии

- TypeScript
- Next.js
- Tailwind CSS
- shadcn/ui

## Локальная разработка

1. Убедитесь, что у вас запущены все необходимые сервисы (API, Redis, PostgreSQL и т.д.) с помощью `docker-compose up` или они запущены локально.
2. Перейдите в директорию `apps/website` и установите зависимости:

```bash
cd apps/website
bun install
```

3. Создайте файл `.env` в корне директории `apps/website` и добавьте переменные окружения из `docker/env-example/website.env`.
4. Запустите приложение в режиме разработки:

```bash
bun run dev
```

Приложение будет доступно по адресу `http://localhost:3000`.
