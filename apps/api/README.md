# api

Это API для проекта, которое предоставляет эндпоинты для аутентификации, управления резюме и выдачи рекомендаций по вакансиям. С ним взаимодействует фронтенд-приложение на Next.js и `llm-worker` с помощью механизма Redis pub/sub для оповещения о новых подборках и отправки события на фронтенд (Server Sent Events).

## Технологии

- TypeScript
- Bun
- Hono
- PostgreSQL
- Drizzle ORM
- BullMQ
- Redis

## Локальная разработка

1. Убедитесь, что у вас запущены все необходимые сервисы (Redis и PostgreSQL) с помощью `docker-compose up` или они запущены локально.
2. Перейдите в директорию `apps/api` и установите зависимости:

```bash
cd apps/api
bun install
```

3. Создайте файл `.env` в корне директории `apps/api` и добавьте переменные окружения из `docker/env-example/api.env`.
4. Запустите приложение в режиме разработки:

```bash
bun run dev
```

Приложение будет доступно по адресу `http://localhost:5000`.
